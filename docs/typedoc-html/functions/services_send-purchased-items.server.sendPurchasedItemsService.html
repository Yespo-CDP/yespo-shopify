      Sends purchased items event data to the tracking service based on the given webhook payload and shop.

      This function retrieves event data for the cart token, constructs the customer and product data, and sends the purchase event to the external tracking API using the shop's API key. If required data is missing (event data, API key, or site ID), the function returns early without sending.
